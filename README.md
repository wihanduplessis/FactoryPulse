# FactoryPulse

A cloud-native manufacturing analytics platform built with .NET and Azure.

## Goals

- Learn Microsoft Azure
- Learn Docker
- Learn CI/CD
- Demonstrate modern .NET backend development
- Build a portfolio project

- ASP.NET Core
- Angular
- SQL Server / Azure SQL
- Azure App Service
- Azure Functions
- Azure Storage
- Docker
- GitHub Actions
